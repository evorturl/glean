# Security and Password Standards

All employees must use the company password manager to generate and store unique credentials for every external service.

Multi-factor authentication is mandatory for all company-managed applications that support it. Employees may not share passwords, recovery codes, or hardware security keys with coworkers.

When traveling or working remotely, employees must connect through the VPN before accessing internal tools on public or hotel Wi-Fi. Sensitive documents should not be downloaded to unmanaged devices.

Suspected phishing, malware, or account compromise must be reported to Security within 30 minutes using the incident hotline or the dedicated security channel.
